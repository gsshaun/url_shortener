URL Shortener Web Application

It's a personal project.

1. The application can check if the inserted URL is a valid URL or not.

2. If valid it can add the URL into the database. And return a short URL to the user.

3. Upon inserting the short URL in the address bar, the application can check if the inserted short URL is a valid one or not.

4. If valid the application can retrieve the original URL from the database. 

5. And the user is automatically redirected to the original URL.


Technology Used: Java, Spring MVC, Hibernate, MySQL Workbench, JSP, Eclipse.
